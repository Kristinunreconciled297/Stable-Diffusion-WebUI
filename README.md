## Overview
**Stable Diffusion WebUI** (by Automatic1111) is the most feature-rich and user-friendly web interface for running Stable Diffusion locally. It allows you to generate high-quality images from text prompts with a wide range of customization options.

> [!Important]
> This is a local AI tool. Requires a powerful GPU (NVIDIA recommended with at least 6GB VRAM) for good performance.

## Key Features
- Easy-to-use web interface
- Text-to-Image and Image-to-Image generation
- Advanced prompting with negative prompts
- Multiple models support (SD 1.5, SDXL, Pony, etc.)
- Extensions system for extra functionality
- Built-in upscaler and face restoration
- Batch generation and grid creation

## Installation
### For Regular Users
1. Download and extract the archive
2. Run `webui-user.bat` (first launch will download models)
3. Wait for the setup to complete
4. Open the link shown in the console

---
## Troubleshooting
| Problem                    | Solution                                      |
|----------------------------|-----------------------------------------------|
| Installation stuck         | Check your internet connection and GPU drivers |
| Out of memory error        | Use lower resolution or enable `--lowvram`     |
| Slow generation            | Use a smaller model or enable xFormers        |
| Antivirus warning          | Add the folder to exclusions                  |
| Black screen / crash       | Update your GPU drivers (NVIDIA Studio recommended) |
